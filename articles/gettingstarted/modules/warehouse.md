<h1 id="supplychain365-supplychain365-features-warehouse-execution">SupplyChain365 SupplyChain365 Features: Warehouse execution</h1>

<p>Introduction to Warehouse execution
The SupplyChain365 Warehouse and Logistics solution consists of the Extension and the mobile application.</p>
<p>Mobile App
Use SupplyChain365 mobile App on Android mobile scanning device to perform warehouse activities.</p>
<p>Mobile App WMS settings
Set parameters for each warehouse activity and automate the logistics flow per warehouse location.</p>
<p>Serialnumbers / Lotnumbers
Register serial and lot numbers using barcode scanning or RFID on the mobile device with the SupplyChain365 WMS APP.</p>
<p>SupplyChain365 WMS workflows
Automate warehouse transactions and automatically create documents and warehouse activities by arranging a workflow and providing parameters.</p>
<p>Break-bulk
Change the packaging unit in the logistics process (from pallets to boxes from boxes to pieces) by using the break-bulk function.</p>
<p>Create License Plates
Create license plates (logistics units: pallets/containers, etc.) and move large amounts of stock on entry, storage and removal with one barcode scan.</p>
<p>Change location
Create settings by warehouse/branch assign warehouse users and easily switch between locations.</p>
<p>Receive
Receive of goods and automatic booking of warehouse receipts including Licenseplates, serial and lot number registration. Input by keyboard manually or scanned by RFID or barcode.</p>
<p>The mobile process can be adjusted on location and/or document with the following settings: Require receive item scan, Scan receive bin first, Batch Posting, Require scan every item, Allow over-receipt, Enable scanning of serial numbers, Use Suggested CrossDockBin in receiving, Scanning of a serial number is required, Enable license plate recording.</p>
<p>Put-Away
Storage of goods in the warehouse using barcode scanning. Register which goods are placed at which storage locations (bins). Input by keyboard manually or scanned by RFID or barcode.</p>
<p>The mobile process can be adjusted on location and/or document with the following settings: Place all Items, Enable scanning of serial numbers, Always automatically create a put-away, Create a put-away when receipt complete, Different Bin confirm required, Scanning of a Serial number is required.</p>
<p>Order picking
Process pick jobs using the SupplyChain365 WMS APP on mobile devices with barcode scanners. Input by keyboard manually or scanned by RFID or barcode. The mobile process can be adjusted on location and/or document with the following settings: Auto open Header, Auto open Line, Place all Items, Enable scanning of serial numbers, Different Bin confirm required, Scanning of a serial number is required, Breakbulk flow required.</p>
<p>Zone / Batch / Tote picking
Collect multiple orders in one run and place items directly in shipping boxes or crates with one or more boxes/crates per order. Input by keyboard manually or scanned by RFID or barcode.</p>
<p>Movements / Replenishment
Move inventory in the warehouse, process replenishment tasks with the SupplyChain365 WMS APP. Input by keyboard manually or scanned by RFID or barcode. The mobile process can be adjusted on location and/or document with the following settings: Auto open Header, Auto open Line, Place all Items, Enable scanning of serial numbers, Different Bin confirm required, Scanning of a serial number is required</p>
<p>Ad hoc movements
Register ad hoc movements with the SupplyChain365 WMS APP and process them automatically. Input by keyboard manually or scanned by RFID or barcode.</p>
<p>Shipments
Book warehouse shipments and print labels and packing slips with the SupplyChain365 WMS APP. Input by keyboard manually or scanned by RFID or barcode.</p>
<p>Loading/unloading
Use loading lists when WMS works with TMS for loading and unloading shipments. Input by keyboard manually or scanned by RFID or barcode.</p>
<p>Inventory/Counting
Create periodic inventory counting lists and check the stock status using the barcode scanning and the SupplyChain365 WMS APP. Input by keyboard manually or scanned by RFID or barcode.</p>
<p>Bin content
Check the contents of a storage location (bin) using the SupplyChain365 WMS APP.</p>
<p>Article availability
Search for items at warehouse storage locations by scanning item number or barcode with the SupplyChain365 WMS APP.</p>
<p>Adjust units of measurement
Add units of measure to items or adjust units of measure. For example: length, width, height, weaving, packaging.</p>
<p>Kitting â€“ make/break Licenseplate
Use the mobile app to pack items together from a single order or to repack items in new packaging. The consumption of packaging material and a new license plate can be activated.</p>
<p>Printing labels
Print barcode/RFID labels in any warehouse process of items, orders, order lines, serial numbers, lot numbers and license plates.</p>
<p>Connect Bluetooth scanning devices
Use external Bluetooth scanners and connect them to the SupplyChain365 APP on the mobile device.</p>
<p>Block Items, Serialnumber &amp; Lotnumbers
Items, Serialnumbers and Lotnumbers may be blocked for further logistics handling (picking, shipping, receiving etc.)</p>
