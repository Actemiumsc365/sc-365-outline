<h1 id="supplychain365-app-component">SupplyChain365 APP component</h1>

<p>TABLE OF CONTENTS</p>
<p>Login
Warehouse functions
Transport functions
Defects
User and group management
Permission sets
Resources
Time registrations
Schedule activities
Version management
Login
You can log in with an AzureAD user. In this way, a user can login via the latest security options and this simplifies management. MFA can also be enabled.</p>
<p>Warehouse functions
Use SupplyChain365 mobile App on Android mobile scanning device to perform warehouse activities.</p>
<p>Transport functions
Use SupplyChain365 mobile App on Android mobile scanning device to perform transport activities.</p>
<p>Defects
Register workplace defects in every logistics process with the SupplyChain365 APP. Add photos and descriptions for further handling in the back office.</p>
<p>User and group management
Group warehouse or transport employees. For warehouse groups it is possible to assign activities to a group. (e.g. group of reach truckers for storage tasks and clearing)</p>
<p>Permission sets
It is possible to assign preference sets to user groups. With the help of these permission sets it is possible to block functionality for certain users.</p>
<p>Resources
Assign warehouse and transport workers to resources so that resource planning and costs become transparent for all transport and warehouse activities.</p>
<p>Time registrations
Record time registrations per warehouse employee, warehouse activity and document. For transport the consignments and trip activities are recorded.</p>
<p>Schedule activities
It is possible to schedule specific orders for a certain time so that they do not become visible to the employee or specifically to a certain employee or group in the mobile application before a certain date or time.</p>
<p>Version management
It is possible to view the actual version of the mobile app. Besides that the app checks if all the necessary versions are available that are in use. If the backend required a new version of the app there will be a message shown.</p>
