    Installing the SupplyChain365 mobile app

    Mobile device and application

    This article cover the following issues:

    • Adding mobile devices
    • deleting mobile devices
    • editing mobile device
    • connecting mobile device
    • and copying installer file to mobile device. If you want to install the mobile application on your Android device go to the admin menu and select Downloads. Then select the APK file SupplyChain365, which is an installer file for Android system.

    Step-by-step application installation

    • Go to portal.supplychain365.net
    • Click on supplychain365 application (APK)
    • Connect your mobile device to laptop via USB cable and transfer the application
    • Click on the application (APK) and install
    • Open the SupplyChain365 application

    Downloading the SupplyChain365 mobile application:

    • If you want to install the application on your Android phone, go to admin menu and select Downloads.
    • Then select SupplyChain365-App (APK file), which is an installer file for Android.
    • It will appear in your downloads folder.

    Next step is to move it to your Android device:

    • First, connect your mobile device and turn on USB debugging mode.
    • Then, copy the SupplyChain365 APK file to your mobile device and select the application to start the installation process.
    • Now, open the application on your mobile device and wait until the screen is loaded. Once you see the screen, use the scan button on your device to scan the QR-code. In a few seconds, the login screen appears .
    • Then, login with your credentials.
    • When a new device is added successfully, you can see information about the device, like its name, brand and model.
    • In the SupplyChain365 portal you can edit and delete the devices that have been connected.

    Mobile users can download the application from the Supplychain365 Portal or in the Play Store.
