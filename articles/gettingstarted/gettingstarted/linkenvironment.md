<h1 id="how-to-link-environment-to-supplychain365">How to link Environment to SupplyChain365</h1>

<p>How to link your environment to SupplyChain365</p>
<p>Now the SupplyChain365 extension is installed. You can also install supplychain365 extension from Microsoft app source website.</p>
<p>The next step is to link SupplyChain365 to your Business Central environment. In your Microsoft Edge browser, open a new tab and go to <a href="https://portal.supplychain365.net/">https://portal.supplychain365.net/</a>.</p>
<ol>
<li>Login with your Microsoft work account. Only a user with admin rights can link environments.</li>
<li>Go to the admin menu &gt; Users and grant permission on behalf of your organization. You only have to do this once in the Users page. Karel can import the office users of the organization. As you can see, Karel is an admin user.</li>
<li>Now go to the admin menu. Select environment and click Register environment. A link code will be generated. Next,</li>
<li>click ‘Copy to clipboard’. Go to the business Central window and open the SupplyChain365 Role Center.</li>
<li>In the Supply Chain 365 Role center, go to setup in the main menu and click Link Portal.</li>
<li>Now paste copied link from clipboard by pressing CTRL + V, linked successfully</li>
</ol>
<p>now you can use SupplyChain365 in business central.</p>
