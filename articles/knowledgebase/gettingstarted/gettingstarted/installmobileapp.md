<h1 id="installing-the-supplychain365-mobile-app">Installing the SupplyChain365 mobile app</h1>

<p>Mobile device and application</p>
<p>This article cover the following issues:</p>
<ul>
<li>Adding mobile devices</li>
<li>deleting mobile devices</li>
<li>editing mobile device</li>
<li>connecting mobile device</li>
<li>and copying installer file to mobile device.
If you want to install the mobile application on your Android device go to the admin menu and select Downloads. Then select the APK file SupplyChain365, which is an installer file for Android system.</li>
</ul>
<p>Step-by-step application installation</p>
<ul>
<li>Go to portal.supplychain365.net</li>
<li>Click on supplychain365 application (APK)</li>
<li>Connect your mobile device to laptop via USB cable and transfer the application</li>
<li>Click on the application (APK) and install</li>
<li>Open the SupplyChain365 application</li>
</ul>
<p>Downloading the SupplyChain365 mobile application:</p>
<ul>
<li>If you want to install the application on your Android phone, go to admin menu and select Downloads.</li>
<li>Then select SupplyChain365-App (APK file), which is an installer file for Android.</li>
<li>It will appear in your downloads folder.</li>
</ul>
<p>Next step is to move it to your Android device:</p>
<ul>
<li>First, connect your mobile device and turn on USB debugging mode.</li>
<li>Then, copy the SupplyChain365 APK file to your mobile device and select the application to start the installation process.</li>
<li>Now, open the application on your mobile device and wait until the screen is loaded. Once you see the screen, use the scan button on your device to scan the QR-code. In a few seconds, the login screen appears .</li>
<li>Then, login with your credentials.</li>
<li>When a new device is added successfully, you can see information about the device, like its name, brand and model.</li>
<li>In the SupplyChain365 portal you can edit and delete the devices that have been connected.</li>
</ul>
<p>Mobile users can download the application from the Supplychain365 Portal or in the Play Store.</p>
