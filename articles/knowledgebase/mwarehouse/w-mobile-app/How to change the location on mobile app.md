# How to change the location on mobile app

How to change warehouse or location.

* click on hamburger menu and select, change warehouse.
* this allows you to switch between warehouses. For example: **EASYBOX, COREBOX, SKUBOX or TRANSBOX.**
* At the top of the screen, we can see that we are in **COREBOX**, which is why this warehouse is not visible in this list. When you switch to EASYBOX, only the processes in this warehouse will be displayed, including numeric indicators: receipt, put-away etc.
