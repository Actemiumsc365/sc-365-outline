# Assembly order (mobile app)

**Assembly order (mobile app)**

Once you have created an assembly order in supplychain365 in Business Central and then a pick from it, now you can operate the following actions in the mobile application:

1. **Register assembly pick order in the APP**
2. **Post assembly order in the app**

***Note***: the process of assembly order differs according to warehouse settings, but if the warehouse setting uses the directed put-away and pick option then you must first pick the order. For non-directed it is enough to ensure that the consumption items are in the correct bin, for example, an inventory movement.

**To register an assembly pick order, you need to follow these steps:**

First you must create a pick order from assembly order then jump to mobile application and follow the next steps. this is how you create pick order.

Open the mobile application open the warehouse main menu and click on the picking process

Click the assembly order that you want to pick, for example, I select this pick order and I know that this pick order is an assembly order.

Open them if they are more than one order click on them and follow on-screen instructions to complete the picking order process.

For more information about picking click here.

**Post assembly order**

After completing the picking order process go back to the warehouse main menu and select the assembly process.

Here are all assembly orders that are ready to be posted, click on the order that you want to post and click on the done button.

Select as much as you want to assemble complete the tracking process and select lot number or serial number if they have then click on the complete button to close the process and post the assembly order.
