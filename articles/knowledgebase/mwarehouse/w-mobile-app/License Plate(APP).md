# License Plate(APP)

**What is a license plate?**

A license plate is a number assigned to each containment unit, which allows for better grouping and management of inventory.

**License plate in supplychain365 mobile application**

Open the supplychain365 mobile application and log in with your credentials (username and password).

See how to log in to supplychain365 tutorials

Click on the license plate process in the main menu, in this menu we have the following processes:

1. **Create a license plate:**click on Create LP enter how many license plates you created and click ok, I have 5 license plates created, this screen asks if you would like to close the popup or print the license plates.                                                     Note: by printing the license plate number you can use them on pallets or keep a record for yourself.
2. **Print license plate:**click on the print license plate to print a specific license plate, here you can enter the license plate number manually or simply scan the license plate barcode by a barcode scanner. After scanning information about the license plate number, identifier, location, and description appears and this is the printed version of the current license plate.
3. **Add or remove items from a license plate:** click on LP items plus or minus to add or remove items, and scan or enter the license plate number to see its information, once the information appears, regarding a specific license plate you can see license plate content here, we have items or other child license plates in this section if it is present then you see then here.  
   here you can select the item or license plate if you want to remove them from the current license plate just enter how many of the items you want to remove, select lot or serial if available then choose bin and click enter.
4. **Delete license plate:**click on the delete a license plate button to remove a specific license plate by scanning the bar code or manually entering the license plate number. Once you find the license plate you see a confirm popup then click yes if you want to delete the license plate or click no if you mistakenly select the license plate.

Note: you can delete license plates that have the states of empty or new and if a license plate has an item or other child license plate first you must remove them and then delete the license plate.
