# SupplyChain365 Mobile Workflow

**SupplyChain365 Mobile Workflow**

**What is a Mobile workflow?**

Workflow automation refers to **the use of software to complete some tasks and activities, without the need for human input**. It's a tool in the supplychain365 extension for making work faster, easier, and more consistent. By automating workflows, supplychain365 reduces the need for manual work and repetitive tasks.

**SupplyChain365 mobile workflow:** it is possible to automate and force tasks by mobile workflow, for example, take a photo automatically in the mobile app when you a specific process or show a message when you enter the warehouse main menu page, and so on.

**How to activate mobile workflow in the supplychain365 extension**

Go to Business Central click on the search button, type mobile workflow, and click the related button.

follow these steps:

1. On the mobile workflow page click on new to add a new workflow
   1. ***Code:*** type the short name of the workflow.
   2. ***Description:*** Describe what this workflow is going to do.
   3. ***Warehouse group:*** select a warehouse group or location for the workflow.
   4. ***Enable:*** this option should be enabled when mobile workflow steps are completed.
2. **Mobile workflow steps:**several fields must be filled with the correct condition and responses for correctly using mobile workflows.
   1. ***Order:*** set the order of workflows if there exist several workflows.
   2. ***When event:***in this field you can add the event which is the base of the workflow. For example, open the warehouse's main menu.
   3. ***On condition:***default value is always but you can define for other values like user, etc.
   4. ***Then response:*** here is the response for example a message or opening another page or screen. So I write a message then let's see what happens to the mobile app when I open the process.
