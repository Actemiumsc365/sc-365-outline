# SupplyChain365 Column Chooser

**SupplyChain365 Column Chooser**

What is a column chooser?

The column chooser **allows a user to change the set of columns at runtime**. It is configured using the column chooser object and may operate in two modes: the default drag and drop mode and the select mode designed for touch devices. Supplychain365 uses the select mode.

Supplychain365 column chooser Important to note that it is possible to choose columns in the mobile app that are shown in the mobile flow. So, it is possible to show the specific columns that are important for a user in for example the receive or pick process.

**How to activate column chooser in the supplychain365 extension**

Go to Business Central click on the search button type warehouse group click the related button and then follow the steps:

1. On the warehouse groups page select a location then on this page click on column chooser or scroll down to find column chooser.
2. select the process in which you want to set column chooser for example receives process
3. click on to column chooser button to open the column chooser page.
4. In column chooser receive page you can add your pre-defined column choosers.
5. Select three dots on the right side of the screen option then select code for example receipt line.
6. In field description select what options will be shown in column chooser.
7. In the label section you can see the abbreviation of the description, so this label is also shown in the mobile app.
8. In the order field you can set the order of the columns.
