# Add or Remove Items or Child License Plate from Licence plate

**Adding Items or Child License plates**

***How to Add items to the license plate and child license plate into the license plate:***

Click on the license plate process in the main menu and select LP items,

First, you must search for or scan a license plate number after scanning it you can see information about the license plate then you can click on add items in here.

**In the take,** page select a bin location or scan the bin location then scan the item or select it by clicking on it, select how many of the items you want to put in the license plate, and click OK, continue this process till you select all item for the license plate then click done button.

**On the place page** select the bin or scan it to put the item in it. Now the operation is successful and here in bin content, we can see the items that we have in the license plate.

So, the process of adding a license plate to a license plate is the same as following the tutorial.

**Removing Items or Child License plates**

***How to remove items from the license plate and child license plate from the parent license plate:***

Click on the license plate process in the main menu and select LP items,

First, you must search for or scan a license plate number after scanning it you can see information about the license plate then you can click on LP content.

**Note:** if the license plate is new or empty then you see a popup message with no content and close the popup.

But if there were items or other license plates inside, then you will see them and remove them from the current license plate.

Enter a license plate number that you know has items or a license plate inside.

Click on LP or license plate content, now click on an item in the remove quantity page enter how many of the items you like to remove, and press enter.

If the item has a serial number or lot number, select it and click Done.

**In the place page,** select bin name and the item removed.

The steps to remove a child's license plate from a parent's license plate are the same as removing the item, follow the tutorial to see how to remove a child license plate.
