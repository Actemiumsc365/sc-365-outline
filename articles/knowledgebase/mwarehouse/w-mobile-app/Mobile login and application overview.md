# Mobile login and application overview

Below there is a video about how to login and a overview of the application.

* Open SupplyChain365 application and wait for login screen to appear.
* Login to mobile device (more information login to Supplychain365 using mobile device)
* Enter username and password.

> These are the QR codes for username and password scan and login. Note: to make your desired QR Code from text or email, search online for QR Code Designer or click here.

* Click on login and verify your account with Microsoft two-factor authentication (If available).
* Select an environment and a sandbox in the form below the supplychain365 logo. Then select company with the demo data and press login.
* The modules are shown. The user can access the warehouse and transport modules regarding their role.
* The numbers in the upper right corner of the blocks indicate the number of open tasks to do.

Extra options that are always avaiable in the mobile app:

* Click on the hamburger menu in the upper left side of screen. Here you will see the following options:
* Change warehouse: this allows you to switch between different warehouses. For example: **easybox, corebox, skubox or transbox.** (for more information click here.)
* App Queue: here you can see all the application queues. By swiping right you can see if the queue has an error. On the right side there is an indication of the status of the queue. It shows whether it is complete, not completed or retry.
* Bin Contents: here you can scan a bin and see its contents or availability. (for more information click here.)
* Item availability: this option allows you to find out how many items are available in the various bins or locations in the warehouse. (for more information click here.)
* Units of Measure: with this option you can scan an item or item code to see information regarding its units of measurement, for example: in box, in piece, height, weight, length. (For more information click here.)
* Select Bluetooth Device: here you can connect a Bluetooth device.
* Logout: log out current user .
* Exit: completely close the application and all login sessions.
* About: displays the copyright information and current version of the software.
